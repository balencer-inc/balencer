# フルテロップ自動化ツール 作成キット

セミナー特典 — 2026年6月11日 / しゅうへい（フリーランスの学校）

このキットは、**喋っている動画にAIが全字幕（フルテロップ）を付けてくれるツール**を、
あなたのPCに **Claude Code が代わりに作ってくれる** ためのものです。
コードは書けなくて大丈夫。AIに「作って」と言うだけで進みます。

---

## 使い方（3ステップ）

### ① 解凍する
このZIPを解凍して、出てきたフォルダを好きな場所（デスクトップ等）に置きます。

### ② Claude Code に渡す
次のどちらかでOKです。

- **解凍したフォルダを Claude Code で開く**（おすすめ）
- または **ZIPファイルのまま Claude Code に渡す**（Claudeが中身を解凍して読みます）

### ③ ひとこと言うだけ
Claude Code にこう伝えてください：

> **「同梱の『AIへの指示_最初に読んで.md』の通りに、ぼくのPCにフルテロップツールを作って」**

あとは Claude Code が、
あなたの環境を5つ質問 → 最小version（文字起こし）→ 校閲画面 → 🎯音合わせ → 書き出し、
までコピペコマンド付きで案内してくれます。**エラーが出ても、原因と直し方をやさしく教えてくれます。**

## 始める前に（ざっくり前提）

- **Claude Code** が使えること（これが主役です）
- パソコン：Apple Silicon Mac（M1以降）/ Intel Mac / Windows のいずれか
- 空き容量 5GB 以上（文字起こしモデルが約1.5GB）
- 仕上げの編集ソフト：Final Cut Pro（Mac）/ Premiere・DaVinci・CapCut（SRTで読み込み）

詳しいインストール手順も Claude Code が順番に案内するので、**今わからなくて大丈夫**です。
