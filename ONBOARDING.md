# ようこそ。ここはバレンサーの「会社の脳」です

このリポジトリ（balencer）は、バレンサーの仕事のやり方・会社の定義・過去の決定が全部入った箱です。
あなたがこれから使う Claude は、この箱を読んだ状態で動きます。だから阿部さんと同じ前提・同じ担当（AI社員）が最初から使えます。

**特別な準備は不要です。** 初期設定（Claudeデスクトップの用意）は阿部さん側で済ませてあります。あなたは下のとおり話しかけるだけです。

---

## まずこれをやってみて（3分）

Claudeデスクトップアプリの「Code」タブでこのフォルダ（balencer）が開いている状態で、チャットにこう打ってください。

```
/secretary おはようございます。今日から使い始めます。何ができるか教えてください。
```

秘書AIが受付して、できること・担当（AI社員）の一覧を案内します。ターミナル（黒い画面）は触らなくて大丈夫です。

---

## 担当（AI社員）の呼び方

やりたいことに合わせて、頭に次を付けて話しかけます。

| 打つ言葉 | 担当 | こんな時 |
|---|---|---|
| `/secretary` | 秘書 | 何から手をつけるか整理／予定・タスク・議事録・連絡文 |
| `/marketing` | マーケ | SNSの下書き・トレンド・note・動画台本 |
| `/consulting` | コンサル | 提案書・クライアント分析・戦略・MVV |
| `/creative` | クリエイティブ | コピー・SNS企画・指示書の叩き台 |
| `/finance` | 経理 | 売上・経費・請求チェック・見積・補助金 |
| `/dev` | 開発 | LP・自動化・API連携・仕様書 |

迷ったら `/secretary` に相談すれば、合う担当へ振り分けてくれます。

---

## たった1つの大事なルール

**会話で終わらせない。作ったもの・決めたことは「箱（このリポジトリ）」に残してコミットする。**

- あなたが作った成果物や、決まったことは、Claudeがこの箱の正しい場所に保存し、「コミット＆プッシュします」と言って保存します。
- 保存されると、翌日には阿部さんや他のメンバーも同じものを使えます。これが会社の資産が貯まる仕組みです。
- チャットの中だけで終わると、他の人には残りません。「これ残しておいて」と一言添えればOKです。

## 作業の置き場所（Claudeが自動でやってくれるので参考まで）

用件はまず4つの箱のどれかに入ります。

- 🏢 バレンサー自社のこと … `docs/company/` ほか
- 🔗 TSUGIの仕事 … `docs/tsugi/`
- 🤝 顧客の仕事 … `docs/clients/<顧客名>/`
- 💡 まだ決まらない思いつき … `docs/inbox/`

## 「3回出たらスキルにする」

同じ作業を3回以上やったら、それは「スキル（AI社員の新しい定型技）」にできます。
「これ毎回やってる気がする」と思ったら、金曜のふりかえりで挙げてください。阿部さんがスキルとして箱に登録し、次からは全員がその技を使えるようになります。

## ログインについて

**会社の仕事は、チームアカウント（バレンサーのチームプラン）でログインして使ってください。** 個人で契約しているプランは私用のみに。

## 下ごしらえは Cowork / Notion でもOK

調べ物・下書き・Notionやドライブの読み書きは、Coworkやチャットでやっても構いません。
ただし **仕上がったものは必ずこの箱（Code）に戻して保存**します。Cowork・Notionは「作業台」、この箱が「倉庫」です。
