# チラシ・1枚もの営業物のレイアウト骨格

旧Notionのチラシ案から「レイアウトの枠」だけを救済。**旧版の文言は流用しない**（下のNG参照）。
経営者に配る／セミナーで渡すA4両面などの1枚もの営業物の版下設計に使う。

## コンサル用チラシ（A4両面）の枠
**表面**
- メインキャッチ（1行）＋サブキャッチ/リード（2〜3行）
- 提供の柱（3〜4点）※看板2枚（組織開発・MVV × AI実装）に沿わせる
- こんな企業・経営者におすすめ（3〜4点）
- 問い合わせ（Web／Email／Tel）

**裏面**
- 進め方（4ステップ：ヒアリング・課題抽出 → 戦略立案 → 実行支援・伴走 → 検証・改善）
- 選べるプラン（現行の階段 ENTRY／CORE／PROJECT に合わせる。詳細は別プラン資料）
- 実績・事例（実績は 64社・生涯4.03億 のみ。個別事例は実在かつ匿名化）
- 会社概要／依頼フロー

## 講演・研修チラシ（A4両面）の枠
**表面**：メインタイトル＋リード／主なテーマ例（3〜4）／講演形態・対象者・実施方式
**裏面**：特長（3点）／講師プロフィール（阿部貴之・著書『社長のためのコンサル3.0』）／実績／ご依頼の流れ・問い合わせ

## NG（旧チラシから絶対に持ち込まない）
- 「経営・組織・デジタルを全部サポート」等の**包括トーン**（単体受注しない・看板2枚で語る）
- **創作事例**（製造業300名50億／ITスタートアップ50名5億 等は実在しない）
- 出どころ不明の実績数字（参加100名・満足度95%・延べ200名 等）
- 旧プラン名（ライト／スタンダード／プレミアム）→ 現行 ENTRY／CORE／PROJECT
- 色・フォント（CI）は現行 `docs/handoff/`・`docs/company/` を正とする
