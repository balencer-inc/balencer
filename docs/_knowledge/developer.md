# 開発 ナレッジ

設計判断・規約・ハマりどころ・依存・デプロイ知見を記録。

## スタック前提
- 本体サイト: Next.js 16 App Router / React 19 / Tailwind 4 / TypeScript / microCMS / Vercel。
- outreach: Next.js + Supabase（`apps/outreach/supabase/migrations/`）。
- Lint `npm run lint` / Build `npm run build` / Dev `npm run dev`。
- GitHub操作は `mcp__github__*`（gh CLIなし）。対象リポ `balencer-inc/balencer`。
