# 1st PLACE株式会社（1stplace）

- **何者か**: 音楽を軸にしたエンタメ／IP企業。音楽・映像・3DCG・音声合成ソフトの企画制作販売、IP（キャラクター/小説/コミック/アニメ/ゲーム）の開発・運用、アーティスト/クリエイターのマネジメント。ヴァーチャルアーティスト「IA」やVOCALOID/CeVIO音声合成ソフト（IA/OИE）で知られる。2004年3月設立・東京都港区赤坂・代表取締役社長 村山久美子・資本金1,000万円・日本音楽出版社協会加盟
- **当社との関係**: 休眠掘り起こしから返信 → Claude での印税分配自動化が本命テーマ（＝音楽著作権/原盤印税の分配業務の効率化。会社の事業内容ではなく当社の支援テーマ）
- **直近の状況**: 村山様と打合せ（2026-06-22）、赤坂訪問（06-17 軸）
- **キーパーソン**: 村山 久美子さん（代表取締役社長）
