# 株式会社アダチ音研（adachi-onken）

- **何者か**: 横浜の総合ポピュラー音楽教室。「GUITARの東大」を中核に、ドラム・ベース・ヴォーカル・ピアノ（鍵盤技塾）・作編曲・レコーディング等の学科をマンツーマンで運営。自社防音スタジオ13室、市販教則本を使わないオリジナル教材、音楽CD企画制作・イベント・機材貸出も。2000年「安達たけしギター教室」創業→2008年1月法人化。横浜市南区永田東・代表取締役 安達武（あだち たけし／バークリー音大卒）・資本金100万円。「日本一のポピュラー音楽教育機関」を掲げる（※音響機器メーカーではない）
- **当社との関係**: 契約済み。5カ年計画（5本柱×2030年で1.2億）を策定、LP制作を継続
- **直近の状況**: 2026-07-13定例（セミナーFB）、06-25・06-12定例
- **キーパーソン**: 安達 武（あだち たけし）さん（代表取締役）
