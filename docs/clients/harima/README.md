# ハリマ紙器印刷工業株式会社（harima）

- **何者か**: 兵庫県加西市の紙器（パッケージ）メーカー。企画・デザイン→製版→印刷→打抜き・貼り加工までを一貫生産。印刷紙器・化粧箱・ダンボール・取説・ラベル・POP・包装紙など。取引先は家電（パナソニックの乾電池/LED電球化粧箱等）、食品・菓子、飲料、日用品・寝具の各メーカー（FDK・ミズノ・オリオン他）。紙のおもちゃ「ガジャガジャ」等の自社商品も。1957年創業/1965年設立・資本金4,000万円・従業員約41名・東京営業所（日本橋）あり・代表取締役社長 藤中亮太（2023年就任・三代目）
- **当社との関係**: HP制作（藤中さんが窓口。直近は2025年12月で止まり）
- **直近の状況**: 2025-12-24 オンライン、12-08 ハリマHP打合せ
- **キーパーソン**: 藤中 亮太さん（代表取締役社長・三代目）
- **要確認**: 現在のフェーズ（HP制作の再開有無）
