- **ブランドの二択が解けた**：会社サイト（チーム受注）＋藤橋さん個人ポートフォリオ（“顔”）の**両立／使い分け**で合意。スーパークリエイター路線は「将来アリ・段階的」。

- **最重要課題が浮上＝クリエイティブディレクター(CD)不在**。藤橋さん一人ディレクションで受注・単価に天井。求める人物像＝「制作×営業折衝ができる人」。
- **阿部の営業の見立てが採用**：専任営業マンより、CDを置いて代理店の受け口を太くする方が映像会社向き。受注9割が代理店経由。年間目標8,000万。クレボ（主要代理店）立て直し。
