# 株式会社イマムラ（imamura）

- **何者か**: 京都市北区の食品企業。「こだわり。対話する。」を掲げ、銘柄和牛の一頭買い・対面型精肉などで差別化。スーパー「新鮮食品館アイハート」（堀川/西院/出町の3店）と対面型ミートショップ「牛彩家アイハートキムラ」を展開（イマムラとして計7店）。グループ会社 株式会社プロフィットヴィクト（2009年設立）が焼肉店「特撰和牛 一寸法師」（北山店/ホテルエミオン京都店）等の飲食事業を運営。1986年3月設立・代表取締役 今村隆宏（和牛マイスター/バイヤー）・資本金1,000万円
