# 株式会社NAILIT（nailit）

- **何者か**: 大阪のセールスプロモーション／セールスコンサル会社。天然水・浄水型ウォーターサーバーを中心に、ショッピングモール等での催事・対面販売（フィールドセールス）を主力とし、正規販売代理店で全国No.1クラスの販売実績（グループ月間5,000台）。通信/電気/ガス等ライフライン商材も扱う。加えて「売れる仕組み」構築のコンサル（営業戦略・組織づくり・人材育成・評価制度）。2022年9月設立・代表取締役社長 茨木凌・社員30名・資本金300万円・急成長（2024年8月期10億→2025年8月期16億）
- **当社との関係**: 営業ダッシュボード案件。v3壁打ちメモ確定、OCRは Gemini for Workspace 併走で Phase 0 実証（＝当社の支援テーマであり、会社の事業内容ではない）
- **直近の状況**: 2026-05-29 オンライン
- **キーパーソン**: 茨木 凌さん（代表取締役社長）／高橋 亮さん（取締役副社長・COO）
