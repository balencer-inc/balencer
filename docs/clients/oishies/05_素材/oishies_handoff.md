## 1. オイシーズ 現状（ヒアリング済み事実）

- 情報基盤: 社内ポータル（Googleサイト）にPDF・スプレッドシート中心で蓄積中。「一旦ためている」が活用像は未定。
- 格納ルールの迷い: 店舗メニューのマニュアルを、どこまで・誰が（本部か店舗か）格納するか定まらず苦戦。

- 組織特性: 直営・FC・シェアリング加盟店＋多国籍スタッフ（特定技能）。均質な情報伝達が難しい。
- MVV: コンテスト形式（jinjer事例）の策定に関心。ただし策定は社内主導の見込み。人事評価は社内に専門家あり、外部ファシリ不要の空気。
- Google Workspace中心の環境。店舗スタッフはWorkspaceアカウントを持たない可能性が高い。

## 2. コア提案（AI活用マップ ①〜④）

効果が最大の領域と、バレンサーが関わるべき領域を「あえてズラす」のが肝。

| # | 領域 | 経営インパクト | 現状（勝者） | バレンサーの関わり |
|---|---|---|---|---|
| ① | 発注・シフト（需要予測で原価/人件費最適化） | ★★★ 粗利直結 | **勝者ほぼ確定**: HANZO（Goals社）。串カツ田中/福しん/吉野家等が導入。初期設定〜店舗浸透まで専属チーム、既存連携でリプレース不要 | **関与しない。名前を渡すのみ・無償・紹介料も取らない** |
| ② | 加盟店支援AI（シェアリング加盟店向け多言語オペAI） | ★★★ 成長事業を拡張 | **まだ勝者なし**。NetBot等の汎用FC本部向けツールの芽はあるが「事業設計」は空白 | **本命。事業設計から伴走（受託でなく事業共創）** |

| ④ | MVV浸透（決まった理念をAIに住まわせる） | ★ 組織の一体感 | 社内主導。策定は貴社 | 軽く伴走。浸透設計のみ |

### なぜ①は関与しないのか
HANZOがデファクト。導入支援も込みで売っており、バレンサーが挟まる価値ゼロ。「直接お話しになるのが一番、我々は一円も取りません」と言い切る誠実さが右腕としての信頼を最大化。紹介料も取らない。

### なぜ②が本命か
FC本部→加盟店のAI支援は発想自体は登場し始めたが、いずれも「本部の問い合わせ対応を楽にする汎用ツール」止まり。オイシーズのブランドシェアリング事業に固有の文脈（冷凍・粉末提供、1〜3分オペレーション、加盟店は本業継続の副収入）を設計し込み、「加盟店に配るパッケージに、24時間多言語で答えるオペAIが最初から入っている＝商品価値そのもの」にする設計は誰もやっていない。ツールベンダーには構造的に不可能。実装エンジンは既存で可、バレンサーが握るのは「設計」と「内製化」。③の資産を横展開して②へ。

## 3. 技術の道筋

- 2026年1月アップデートで**法人版GeminiからNotebookLMを直接参照可能**に。「社内マニュアルに基づき新人の質問に答えるAI（Gem）」が簡単に構築できる。
- Google環境で「導入ゼロ」で始められる: 新システム導入ゼロ / データ移行ゼロ（ドライブのPDF・docをそのまま参照、更新も反映）/ セキュリティ審査ほぼゼロ（ドライブの既存権限が適用、法人版は外部非開示・非学習）。
- **二層UI**: 本部・SV・人事＝Gemini＋NotebookLM / 店舗スタッフ＝LINE（使い慣れたスマホUI、多言語）。
- LINEは Messaging APIで自社ボット / まずLINE公式AIチャットボット(β)（PDF・画像からQ&A自動生成）で軽く検証、の両道。作り込まず「本部で60分動くもの」を先に体感させる。

### 参考事例（商談で使える）
- 松屋フーズ: 約1,200店舗の問い合わせをAIチャットボット化。FAQをExcel一括登録で1週間で運用開始。LINEキャラで浸透、月1万件以上を自動対応、本社電話問い合わせ約41%減。質問ログで知識を育てる運用。
- 日清食品HD: 社内生成AIで社員一人あたり年79.4時間削減、ROI3.9倍、月間利用率85%（CFO向け数字）。

## 6. 想定問答A（CFO・ビジネス論点／提案ロジック準拠）

- Q いくらかかる? → 全体は今出せない（出す会社は疑うべき）。入口は診断¥50万だけ。先はログを見てから。
- Q トータルは? → 診断¥50万→伴走¥30-50万/月×3ヶ月（研修は助成金圧縮）→店舗250-400万。最初の判断は¥50万で済み大きい数字は後ろ倒し。
- Q 開発会社と何が違う? → 制作会社は「作って納品」で終わる、大手は億でオーバースペック。我々は「社員自身が回せるまで設計」＝抜けても回る。
- Q 途中でやめられる? → 診断は単発で負債残らない。伴走は最低3ヶ月で義務なし。作った資産は御社に残る。囲い込まない。
- Q IT専任いないが回る? → 専任なし前提で設計。Google内完結、店舗はLINEで新規学習ゼロ。AIに詳しい人を社内に作るまでが伴走の中身。
- Q セキュリティは? → 法人版Workspaceで外部非開示・非学習。ドライブの既存権限が効く。追加審査ほぼ不要。
- Q ROIは? → 「何%削減」と断言する会社は疑うべき。松屋の電話4割減が同型。診断で現状工数を数値化しbefore/afterで測れる形で始める。
- Q シェアリング加盟店にも使える?（②の匂わせ） → さわりだけ。「制作費で測る話でなく事業共創。これこそ飲みながら」で会食へ橋渡し。
- Q 持ち帰りたい → 歓迎（押さない）。今日は人となりを見てもらえれば十分。続きは一杯やりながら。
