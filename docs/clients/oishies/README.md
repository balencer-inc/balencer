# オイシーズ株式会社

> ⚠️ 別会社注意: 兵庫の食品メーカー「株式会社オイシス（売上322億・従業員3,300名）」とは**完全に別会社**。混同しないこと。
> オイシーズ株式会社は、**「おいしい」で、世界にイノベーションを**というビジョンを掲げ、日本を代表する食文化を国内外に展開している持株会社です。2016年10月に設立され、傘下子会社の経営支援や海外フランチャイズ（FC）事業を中心にビジネスを展開しています。

会社全体の事業概要は、大きく以下の3つの柱で立体的に捉えることができます。

**1. 国内外での強力な飲食ブランドの展開** 日本を代表する郷土料理である天ぷらと、国民食であるラーメンの人気ブランドを軸に事業を展開しています。グループが擁する代表的なブランドには以下のものがあります。

- **日本橋 天丼 金子半之助**（天丼・天ぷら）
- **つけ麺 つじ田**（濃厚豚骨魚介つけ麺）
- **博多長浜らーめん 田中商店 / 中華そば専門 田中そば店**

これらのブランドを中心に、直営店およびFC店をグローバルに拡大しており、2023年12月末時点での**グループ店舗数は計105店舗（国内80店舗、海外25店舗）**、**グループ売上高は102億円**という規模に成長しています。

**2. 積極的な海外展開と「逆輸入」** 「日本の『おいしい』を世界へ。世界の『おいしい』を日本へ。」というメッセージのもと、アメリカ（ロサンゼルスやヒューストン）、タイ、インドネシアなどへ多店舗展開を行っています。特徴的なのは単なる海外進出にとどまらず、アメリカ・ロサンゼルスで展開しミシュラン（ビブグルマン）にも掲載された担々麺ブランド「KILLER NOODLE」を、デリバリーブランド等として日本へ逆輸入するといった柔軟な展開も行っている点です。

**3. 革新的な「ブランドシェアリングサービス」事業** 実店舗の運営だけでなく、既存の飲食店（カフェや居酒屋、定食店など）の売上向上を支援する**フードライセンスのシェアリングサービス**を展開している点が同社の大きな強みです。

- **仕組み：** 既存の飲食店が自店舗の営業を続けながら、オイシーズが持つ有名ブランド（「つじ田」や「幸ちゃんラーメン」「ぎ郎」など）の商品を、デリバリー、テイクアウト、イートインで提供できるサービスです。
- **加盟店のメリット：** メインの原材料は冷凍や粉末で提供され仕込みが不要なうえ、1〜3分で完了する簡単調理のオペレーションが組まれています。これにより、人員を追加することなく既存の設備と人材だけで、年間1,000万円以上の売上アップを見込める新たな収益源を飲食店に提供しています。

このように、オイシーズ株式会社は単独の飲食店経営にとどまらず、**強力なブランドポートフォリオを武器にした多角的な店舗展開（直営・FC）**と、**他社の飲食店インフラを活用するブランドシェアリング**という立体的なビジネスモデルによって急成長を遂げている外食企業だと言えます。
