営業フロー

出会いから契約までを1本の線でつなぐ。この流れを、あなたの会社にも。
BALENCERfor loved ones.
01

流入

Reach — 知ってもらう

知らない相手に見つけてもらう入口。ここの量が後ろすべての母数になる。

お願いできること
  • 紹介が生まれる仕組みづくり
  • SNS発信の型と運用
  • ブログ／SEO記事の制作
  • 名刺・イベントのリスト化
02

診断

Diagnose — 現在地を見せる

数分の無料診断で弱点を数字にする。課題が自分ごとになり対話が始まる。

お願いできること
  • 無料診断ツールの設計・構築
  • 設問とスコア判定ロジック
  • 結果レポートの自動メール
  • 診断ページの制作
03

予約

Book — 会う約束を取る

その場でオンライン相談の枠を選んでもらう。日程調整の往復をなくす。

お願いできること
  • カレンダー予約の導入
  • 自動リマインドの設定
  • 社内通知（Slack等）連携
  • リード情報の自動記録
04

クロージング

Close — オンラインで決める

実例をそのまま見せて信頼をつくる。売り込まず、多くは2回目で決まる。

お願いできること
  • 初回相談の設計・同席
  • 見せる実例・デモの用意
  • 提案書・見積の作成
  • クロージング資料
05

コンサル開始

Start — 伴走がはじまる

毎月「使えるもの」を1つ納品しながら伴走。成果物で価値が続く。

お願いできること
  • 契約まわりの整備
  • 月額伴走の設計
  • 毎月の納品物づくり
  • AIでの業務仕組み化
Nurturing — 全ステージ横断
いま決めない人を、切らずに温め続ける。
眠り名刺の掘り起こし交換したまま眠る相手を、許諾を取って配信対象に。
ステップメール診断リードを自動で追いかけ、予約が入ったら止める。
定期メルマガ現場の一次情報を配信。頃合いで相談に戻ってくる。
実証：あるホテル向け支援会社は 紹介→診断→2回目で契約（半年の伴走）。この型で決まりました。
