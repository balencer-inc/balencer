# サービス体系の正本 — サービスの階段（一本化）

> 会社の売り物はこの一本の階段。カテゴリを並列に3つ並べる旧表現（サービス3カテゴリ）は使わない。
> 各商品の詳細（価格表・LP原稿）はそれぞれの置き場が正。ここは「体系と関係」の正本。

---

## 階段の全体像

| STAGE | 商品 | 価格 | 役割 |
|---|---|---|---|
| **ENTRANCE** | ENTRY（伴走プラン） | 月20万〜 | 経営の壁打ち相手。裏でAI分析＋毎月「使えるもの」を1つ納品。**入口の伴走プランの正式呼称** |
| | ミタテ／ヒトツ／補助金／TSUGI診断 | 各商品ページ参照 | 出会いの入口 |
| | SOERU | 月10万 / 20万 | 入口の一形態。ただし金額が低く現時点でニーズも限定的なため**当面は控えめに扱う** |
| **CORE** | MAIN 右腕 | 月30〜50万 | 思想は『社長のためのコンサル3.0』。町医者型、レポートで終わらない |
| | PREMIUM 経営参謀 | 月60〜100万 | 右腕の上位。経営の中枢に入る |
| **PROJECT** | MVV策定・浸透プログラム／AI実装／CI・VI／Web・制作 | 案件見積 | 制作は必ず伴走に挟んで提供（制作単体では受けない） |
| **SUMMIT** | AI組織OS構築／採用ブランディング接続／事業承継準備（TSUGI連携） | 案件見積 | 深く入った先にだけ提供する出口 |

- 入口の伴走プランの正式呼称は **ENTRY（月20万〜）**。旧禁止だった「Entryプラン」の呼称ルールは解除された（CLAUDE.md §1）
- **SOERU** は入口の一形態として残すが、当面は控えめに扱う（旧「SOERUに振り切り」方針からの更新）。既存のお客様は何も変わらない
- 補助金は出会いの経路・費用を軽くする手段（会社の看板ではない）
