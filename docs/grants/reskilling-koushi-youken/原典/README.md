## 置いたあとの流れ

1. このフォルダにファイルを置く（ファイル名は日本語のままでOK）
2. **ローカルCLIのClaude**なら、私が `git add → commit → push` まで実行します
3. **Web/クラウドのまま**進めたい場合は、ローカルで `git add docs/grants/reskilling-koushi-youken/原典/ && git commit && git push` していただければ、私が `git fetch` で取り込みます

ファイルが入り次第、私は次を行います：
- 計画届の内容を精読し、(a)様式記載ドラフト／(b)職務経歴書／(c)カリキュラム骨子との**整合チェック**（訓練名・科目・講師名・訓練日/時間・場所の一致）
- 様式第10-2号の原典が入れば、**年数・定義（3年/10年・当該分野）を逐語ベースで確定**し、各書類の「要・原典確認」を解消
- 形式不備リスクの最終チェック
