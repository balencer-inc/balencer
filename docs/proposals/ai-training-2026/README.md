# AI研修 提案資料 一式（2026）

クライアントが「AI導入補助金でClaude研修を申し込む」と言っていた件の交通整理から生まれた提案パッケージ。
**研修そのものへの補助は AI導入補助金（経産省）ではなく、リスキリング助成金（厚労省）が受け皿**。これを軸に、研修を入口受注し、後段は受託開発（本命）またはSOERUへ引き上げる。

**※ 1st PLACE株式会社 村山様向けにパーソナライズ済み**（題材＝印税分配の仕組み化）。他案件で使うときは複製して固有名（1st PLACE／印税分配）を差し替える。

## 商談共有パック（村山様オンライン）
① 本提案デッキ `index.html`（リスキリング研修）＋ ② 匿名事例 `case-study.html`（業務自動化の型）の2点を共有。
②は「Notion中心の例。御社はClaude中心で、裏のDBは適材適所」と位置づけを添える（Notion導入の話と誤解させない）。NAILIT実名資料（`docs/estimates/nailit-2026/`）は社外提示しない。

## 確定方針

- 形式：HTML提案デッキ
- プラン：本命1プランに絞る（松竹梅にしない）
- 差別化：Claude特化 × 研修後の自動化実装まで伴走（入口は汎用AI研修として入りやすく）
- 提出書類：カリキュラム表＋見積書の2点見本

## 金額（税抜・10名想定）★要最終確認

- 研修本体（OFF-JT 15h）… **¥1,000,000**（助成金の対象）
  - 助成適用後の実質負担 目安 **約10〜25万円**（経費助成60〜75%＋賃金助成1,000円/h×15h×10名≒15万）
- 研修後の展開（助成金の対象外・別建て）— 後段の出口は2つ
  - **受託開発**（印税分配の仕組み構築＋保守）… 要件定義後にお見積もり（金額は資料に出さない）
  - **SOERU**（月額AI顧問）… ¥100,000/月〜（継続伴走が要る場合の受け皿）

> 「実装伴走3ヶ月」案は中途半端なので廃止。後段は受託開発（本命）／SOERUへ。受託は1st PLACEのステートメントのサンプル入手→要件定義後に見積。人数・回数は案件確定時に調整。

## 助成金の要点（提案で守る前提）

- 制度名：人材開発支援助成金（事業展開等リスキリング支援コース）／厚労省・労働局
- 中小：経費 基本60%〜最大75%、賃金助成 1,000円/h、OFF-JT 10時間以上
- **計画届は研修開始の1ヶ月前まで**（1日遅れで全額不支給）
- 制度は**2026年度（令和8年度）まで**。来年度以降未定
- 入金まで6〜8ヶ月
- **申請代行はしない／できない**＝社労士の専権業務。バレンサーは研修＋添付書類（カリキュラム表・見積書・講師要件確認書・出席簿・実施報告書）まで

## 数字・出典の扱い

- デッキ・見本に載せた数値は**2026年6月時点の調査／公開情報にもとづく参考値**。最終要件・支給可否は管轄労働局の判断。
- 様式番号・加算（設備投資加算等）は労働局で更新があるため、提案では断定せず「最新を要確認」とする。
- 一次情報の主軸：厚労省「人材開発支援助成金」ページ群、各労働局の様式一覧。市場相場は各研修会社サイト・調査記事（二次情報含む）。
