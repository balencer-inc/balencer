# AI研修 提案資料 一式（2026）

クライアントが「AI導入補助金でClaude研修を申し込む」と言っていた件の交通整理から生まれた提案パッケージ。
**研修そのものへの補助は AI導入補助金（経産省）ではなく、リスキリング助成金（厚労省）が受け皿**。これを軸に、研修を入口受注し、業務自動化の実装伴走へ引き上げる。

## 成果物

| ファイル | 用途 |
|---|---|
| `index.html` | HTML提案デッキ（12枚・1280×720）。来週クライアント提示用。soeru-client-deck型を流用 |
| `samples/curriculum.html` | カリキュラム表 見本（A4縦）。計画届に添付。OFF-JT 15hが一目で分かる |
| `samples/estimate.html` | 見積書 見本（A4縦）。研修本体（助成対象）と実装伴走（対象外）を2層で分離 |

## 確定方針

- 形式：HTML提案デッキ
- プラン：本命1プランに絞る（松竹梅にしない）
- 差別化：Claude特化 × 研修後の自動化実装まで伴走（入口は汎用AI研修として入りやすく）
- 提出書類：カリキュラム表＋見積書の2点見本

## 金額（本命1プラン・税抜・10名想定）★要最終確認

- 研修本体（OFF-JT 15h）… **¥1,000,000**（助成金の対象）
  - 助成適用後の実質負担 目安 **約10〜25万円**（経費助成60〜75%＋賃金助成1,000円/h×15h×10名≒15万）
- 実装伴走 3ヶ月 … **¥900,000**（月30万×3・助成金の対象外・別建て）

> 金額・人数・回数は案件確定時に調整。商品名「AI実装研修」は仮称（正式名は別途）。

## 助成金の要点（提案で守る前提）

- 制度名：人材開発支援助成金（事業展開等リスキリング支援コース）／厚労省・労働局
- 中小：経費 基本60%〜最大75%、賃金助成 1,000円/h、OFF-JT 10時間以上
- **計画届は研修開始の1ヶ月前まで**（1日遅れで全額不支給）
- 制度は**2026年度（令和8年度）まで**。来年度以降未定
- 入金まで6〜8ヶ月
- **申請代行はしない／できない**＝社労士の専権業務。バレンサーは研修＋添付書類（カリキュラム表・見積書・講師要件確認書・出席簿・実施報告書）まで

## 数字・出典の扱い

- デッキ・見本に載せた数値は**2026年6月時点の調査／公開情報にもとづく参考値**。最終要件・支給可否は管轄労働局の判断。
- 様式番号・加算（設備投資加算等）は労働局で更新があるため、提案では断定せず「最新を要確認」とする。
- 一次情報の主軸：厚労省「人材開発支援助成金」ページ群、各労働局の様式一覧。市場相場は各研修会社サイト・調査記事（二次情報含む）。
