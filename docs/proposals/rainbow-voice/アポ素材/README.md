# アポ素材フォルダ（Rainbow Voice 初回 2026/7/8）

初回用1枚デッキを作る前の「素材置き場」。阿部さんの補足・画像をここに集約する。

## 使い方
1. `言いたいこと.md` を開いて、書きたいことを直接書き足す（空欄はそのままでOK）。
2. 画像は `images/` に置く（PNG/JPG）。ファイル名は内容が分かる名前に（例: `売上構造.png`）。
3. 使いたい画像は `言いたいこと.md` の該当欄・末尾の表にファイル名をメモ。
4. 「デッキ作って」と言ってもらえれば、このフォルダの中身＋分析メモ（`../初回打ち合わせ-分析メモ.md`）を統合して3画面デッキにします。

## 中身
- `言いたいこと.md` … 阿部さん記入用テンプレ（デッキ3画面に対応）
- `images/` … 画像を置く場所
- 参照元: `../初回打ち合わせ-分析メモ.md`（分析本体）
