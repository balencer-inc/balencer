# 事例：アダチ音研（GUITARの東大）｜金丸さん初回で使う"生きた実例"

> 出典: Notion「アダチ音研様 コンサルティング報告書」(2026/3)・「戦略策定シート」。
> なぜこの事例か＝**金丸さんと構造がそっくり**（個人メソッド依存の教育業→仕組み化・型化・オンライン展開・FC化で伸ばす）。紹介元本人の事例でもある。

## 会社像（Before）
- 「GUITARの東大」ブランドでギター教育。横浜拠点、オンラインで全国展開を志向。
- 講師8名・レッスン生 約200名。安達社長の強烈なメソッド（フィジカル＆ナレッジ法）と理念（音楽を教えるプロをつくる／日本一の教育機関）。
- 自己診断: **成長準備型 72/100・デジタルに課題**。
- 痛点: **SEO会社にマーケを外注していたが効果が出ない**。強み・理念はあるが、集客と展開が個人・属人に依存。

## バレンサーがやったこと（2年・5フェーズ）
| Phase | 期間 | テーマ | 中身の要点 |
|---|---|---|---|
| 1 | 2024/4-6 | 戦略策定・現状分析 | **STP再定義**。ポジショニング＝「オンラインで全国に顧客→リアル総本山の価値が上がるグラデーション」。カスタマージャーニー×顧客ピラミッドでKPI設計 |
| 2 | 2024/7-9 | SNS施策・ウェビナー | TikTok=入口／YouTube=収益、の棲み分け。ショート月15本・火曜21時ライブ。初ウェビナー本番(7/31) |
| 3 | 2024/10-2025/3 | マネタイズ構築 | **8つの収益柱**（1対多レッスン/サブスク/教材/ウェビナー/YouTube/1対1プレミアム/出張/コンサル）。5カ年売上試算 |
| 4 | 2025/4-9 | 業務自動化・制度設計 | **レーティング（8段階）＋ライセンス（生徒→講師のキャリアパス）**。**認定教室＝FC**（ロイヤリティ月5万）。契約書10種・IP整理。報酬計算/決済(Stripe)自動化 |
| 5 | 2025/10-2026/3 | LP・メンバーシップ | YouTube連動メンバーシップ（聴講¥490/実習¥990/研究¥1,790＋オンラインレッスンZ ¥14,680〜）。ブランドGL（タグライン「ギターは学問である。」） |

## 効果・設計値
- **収益の階段化**: 無料入口(SNS/テスト) → 少額サブスク(¥490〜) → 本命(¥14,680〜) → 講師化(月+3,000〜15,000円) → 認定教室FC(ロイヤリティ月5万)。
- **5カ年試算**: 1年目 1,440万 → 5年目 2,940万（年成長20%、認定教室10→20）。
- **属人からの脱却**: 安達個人の指導 → レーティング/ライセンスで"教える人を増やす仕組み"へ。個人依存を制度で外す。

## → 金丸さん（Rainbow Voice）への写像（デッキ③の核）
| アダチ音研で解いたこと | 金丸さんでの対応（同じ設計図） |
|---|---|
| 個人メソッド依存（安達1人） | 金丸メソッド（ハリウッド式）が金丸1人にロック → 型化・言語化で外す |
| SEO外注が効かない | 集客10/25。強み（著書・TV・法人実績）を"言語化→導線"に変換 |
| ポジショニング再定義 | 「ボイトレ屋」→「経営者の声ブランディング／ボイス・ウェルビーイング」へ |
| 収益の階段＋サブスク | 無料入口→サブスク→エグゼクティブ高単価→法人研修、の階段設計 |
| レーティング＝生徒→講師→FC | 認定トレーナー制度でメソッドを再現・展開（一人稼働の天井を外す） |
| ビジョンに沿ったデジタル戦略 | ウェルビーイング×人的資本の旗の下にAIファネル |

**一言**: 「安達さんの会社で2年かけてやったこと（強みを再定義し、仕組みにして、属人を制度で外す）を、金丸さんの"声"のビジネスでも同じ設計図で描けます。」

## 使い方の注意
- 数字（生徒200名・5カ年試算・料金）を出す範囲は初回前に阿部さんと確認（`言いたいこと.md` §G）。
- 「3ヶ月以上かけて練り上げた＝だからブレない」は必ず添える（安売り防止・期待値設計）。
