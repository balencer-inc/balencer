## 追補（2026-07-05②）HP精読で判明した重要点と見立ての更新

阿部さん提供のHP分析で、初回の見立てを **上方修正**。

1. **実績スケールが大きい**: **経営者・政治家など2000人以上の指導実績**（累計。〜約10年で年200人ペース）。「無名の一人ボイトレ」ではなく、**実績とブルーチップ顧客を持つ確立ブランド**。
2. **法人B2Bが本物**: **ポルシェEBIグループ、ラグジュアリーブランド、商工会議所**からの企業研修依頼。→ 法人単価が高く、フェルミの法人枠(C)を**上方修正**（下記§2）。
3. **メソッドが体系化されている**: 「声・呼吸・身体・脳・メンタル」を統合。自律神経・脳神経・ストレスケアまで踏み込む。→ 単なる発声指導でなく**再現可能な体系に近づけている**（当初の「型化ゼロ」想定は緩む）。
4. **新ポジショニングが当社のど真ん中**: **「ボイス・ウェルビーイング」を健康経営・人的資本経営の文脈で法人に広げる**方針。理念は「声から人と組織を本来の姿へ導く」「Rainbow＝個性をブランドに」。→ **彼女は既に"ボイトレ屋"から"組織のウェルビーイング/人的資本"へ移動中**。これは**バレンサー看板1（組織開発・MVV）の隣接領域**。
5. **要注意＝多角の拡散**: エグゼクティブ／アーティスト／婚活／ふくし／親子（AIに負けない子ども）／政治家／企業…と**縦が乱立**。診断「立て直し型＝まず1つ」と完全に符合。**焦点の欠如**が最大の弱み。

### 戦略ウェッジ（更新後・初回で刺す一言）
> 「金丸さんは既に答えの方向（＝声で組織のウェルビーイング・人的資本を上げる）を見つけている。足りないのは、その旗の下に**縦を1つに絞り・言葉を統一し・法人に届く導線と仕組みを作る**こと。強みも実績もある。**"活かしきれていない"の正体は、拡散と個人依存**です。」

**絞るなら＝法人の「ボイス・ウェルビーイング／人的資本」**（最高単価・最も防御可能・本人の新ビジョンと一致・当社と最も噛む）。婚活/親子等は理念活動 or 将来オプションとして後ろに置く。

## 1. 何の会社か（事業の骨格）

- **Rainbow Voice株式会社**（レインボウボイス）。設立 2016年10月21日、代表取締役 金丸明日香。スタジオは東京（品川区/渋谷）＋オンライン。
- ブランドは **BRAND VOICE（ブランドボイス）**。旧称 **プラウドボイスメイク®**。キーワードは **「聞き惚れボイス®」「声を作らず自信をつくる」**。
- 手法は **ハリウッド式**（マイケル・ジャクソンのボイストレーナー直伝を掲げる）。歌のボイトレではなく「本来の声を引き出す」思想。
- 権威資産: 2018年に**小学館より著書『人生が変わる!! 本当に「いい声」になる方法』**、NHK BS「美と若さの新常識」「所さん、大変ですよ!」等**TV露出**、昭和医科大学リカレントカレッジ講師、講師派遣会社（システムブレーン/ブレーン）登録。
- 主力の顧客層と商品:
  - **BRAND VOICE エグゼクティブコース** … 社長・起業家・管理職・セミナー講師・研修講師向け（＝高単価・本丸）
  - 一般/個人のマンツーマン（入口 1レッスン **約6,080円**、無料体験あり）
  - 法人研修・講演（「声で人を魅了する方法」等）
  - 発表会イベント（Proud Voice LIVE）、福祉×ボイトレ（高齢者施設ワークショップ）、「世界のみんなで歌おうプロジェクト」など**新規/派生の動き**

### 体制の読み（重要）
- **社会保険 被保険者数 1名** → 実質「一人社長」。**他の講師がいても業務委託**の可能性が高い。
- つまり **売上の中核（エグゼクティブ×権威×本人稼働）が金丸さん個人にロックイン**。委託講師は入口レッスンを回すが、ブランド価値＝「金丸のハリウッド式」で**再現性がないため高単価は本人にしか乗らない**。
- これが診断の **仕組み化 11/25 / 展開できない構造** の正体。**創業者＝製品** の典型。

## 2. 売上のフェルミ推定

**【更新後】結論（現実線）: 年商 約1,500〜2,200万円。レンジで 1,200〜2,800万円。良い年で3,000万に届くか。**
（※初回見立ての1,200〜1,500万から、法人ブルーチップ実績を織り込み上方修正）

「被保険者1名」は依然として上限のアンカー。年商が恒常的に3,000万超なら通常もう1〜2名を正社員化するはずで、それが無い＝**天井は3,000万前後**というサニティチェックが効く。法人研修の実受注ボリュームが最大のスイング要因＝**初回で必ず確認**。

### この数字から言える一番大事なこと
**「稼働×単価」で頭打ち。** 売上の大半が金丸さん1人の時間に固定され、時間は増やせない。だから**「値上げ（高単価化）」か「本人以外でも回る仕組み化」か「集客の再現化」以外に伸びしろがない**。まさに本人が言う「強みはあるが活かしきれていない」の構造的原因。

  - 経営者パーソナルブランディングの個人プロデューサー多数

  1. ハリウッド式×MJのトレーナー直伝という**唯一無二の物語**
  2. 小学館の著書＋TV露出という**第三者権威**
  3. 「聞き惚れボイス®」「プラウドボイスメイク®」の**商標**

## 4. 診断×本人課題×バレンサーの接続（課題化フレーム）

本人の言葉:「ブランディングや強みはあるが、それが活かしきれていない。今後展開したい事業・ビジョンに向けて何が必要か」
→ **バレンサーのど真ん中（＝経営者の右腕・ブランディングAX）。**

### 初回30分で提示する仮説（＝課題化の核）
> 「金丸さんの問題は"強みがない"ことではありません。**唯一無二の強み（ハリウッド式×権威×思想）が、金丸さん個人の稼働にロックインされていて、①正しく言語化されていない ②再現できる仕組み／商品になっていない ③それを届ける集客導線がない**――この3点です。だから、稼いでも増えない・展開できない。」

### バレンサーの3層で整理（インナー→アウター→事業）
1. **インナー（組織を変える）**: 一人依存からの脱却。**「金丸メソッド」を言語化・体系化**し、一部を本人以外でも提供できる型に。MVV/ビジョンの言語化（＝立て直し型「まず1つに絞る」）。
2. **アウター（魅せ方を変える）**: **ポジショニング再定義**。「ボイトレ教室」→「**経営者の"声"ブランディング**」。看板商品の再設計、CI/VI・Web・言葉の統一。→ **強みの言語化が単価と成約率を変える**（D社の型）。
3. **事業（戦い方を変える）**: **エグゼクティブ特化の高単価×少数**設計＋**AIで集客/コンテンツ/ファネル**を回す。SOERUで壁打ちしながら毎月「使えるもの」を1つ納品。

### 診断との接続で刺す
- 立て直し型 49/100 → **「全部やらない。まず1つ」**で安心させる。キーメッセージ **「正しい順番で、1つずつ」**。
- 最低スコア=集客10/25 → 入口の会話は集客/Webから。ただし**本丸はブランド18/25の資産を"言語化→高単価商品化"して売上に変換すること**。
- 事例: **専門商社D社（年商7億）→ 強みの言語化で成約率25%→45%→年商7→10億**。規模は違うが「**言語化が単価と再現性を変える**」実例として提示。

### 提案の落とし所（初回のゴール）
- **最短**: **SOERU（月10万/20万）**で「まず1つ」＝ポジショニング＆看板商品の再定義から着手、毎月"使えるもの"を納品。
- **中期**: MVV/ブランディング PROJECT ＋ AI実装（集客・コンテンツ自動化）。
- 刺し方: **町医者型・レポートで終わらない・実装まで自分たちの手で伴走**。

## 5. 初回30分の質問リスト（限られた時間で課題化する）

事前に用意。「現状の可視化 → 一番の痛点 → 展開したい像」の順で30分に収める。

1. **売上構成の比率**: 本人レッスン／委託講師／法人研修／講演・イベントは、ざっくり何:何? →（一人依存度の確認）
2. **講師体制**: 講師は何名で、どのメニュー担当? 「金丸メソッド」はどこまで型化・マニュアル化されている?
3. **集客の入口**: 今、新規はどこから来ている?（紹介／SNS／検索／メディア／講演）→ 再現性はあるか
4. **一番"利益率が高い＆やりたい"顧客**は?（歌／一般ビジネス／エグゼクティブ／福祉）→ 絞り込みの軸
5. **「展開したい事業・ビジョン」の具体像**は?（多店舗／講師養成／法人研修化／オンライン商品／メソッドのフランチャイズ 等）
6. **一番"活かしきれていない"と感じる瞬間**は?（＝本人が言語化した痛点。ここを掴めば課題化が刺さる）

> ゴール: 6つの回答から「①言語化 ②仕組み化 ③集客」のどこがボトルネックかを本人と合意し、SOERU起点の"まず1つ"に落とす。
