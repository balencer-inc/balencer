株式会社バレンサーは、ブランディングを軸に、AX（AIによる業務・組織の変革）まで一気通貫で全国の企業に伴走する会社です（設立2019年）。ブランド・組織開発・システム開発・AI実装まで、自社とパートナー網で完結します。

faq

よくあるご質問

初期費用はかかりますか？
初期費用は0円です。月額のみで、今日からでも始められます。
なぜ最低3ヶ月なのですか？
AIは一度きりでは定着しません。毎月「使えるもの」を積み上げ、現場に根づかせるために、最低3ヶ月の継続をお願いしています。多くの会社が3ヶ月で「AIが当たり前」の状態に変わります。腰を据えて成果を出したい会社のためのサービスです。
AIにまったく詳しくないのですが、大丈夫？
むしろ、そういう会社のためのサービスです。専門用語は使わず、御社の言葉で進めます。難しい設定はこちらで行います。
ツールの費用は別ですか？
Notion・Make・Claude API などの外部利用料は実費でご負担いただきます（多くは月数千円〜）。何にいくらかかるかは事前にご説明します。
情報やセキュリティは大丈夫ですか？
利用ルール（ガイドライン）づくりもサービスに含みます。情報の扱いを整えてから、安心して全社に広げられます。
誰が担当しますか？
代表の阿部が直接伴走します（内容に応じてメンバーが加わります）。
